[`robodraw`](robodraw) is an ergonomic and programmatic drawing library for
python. It is a wrapper around `matplotlib` that provides a more intuitive way
to specifically create drawings and diagrams, including in pseudo-3d, rather
than plotting data. It provides the backend for the drawing functionality
in `quimb` and `cotengra`.
